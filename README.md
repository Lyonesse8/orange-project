# 🍊 Orange Data Mining Projects

Welcome to my Orange project repository! This collection showcases various workflows and projects I created in my Digital Banking and Fintech class. Each workflow demonstrates different data mining and machine learning techniques using the Orange software.

## 📚 Table of Contents
* bank_marketing.ows - Analysis of bank marketing campaign data to predict customer subscription behavior
* cyber_security_final.ows - Cybersecurity threat classification and anomaly detection workflow
* houseing_price_prediction.ows - Regression-based housing price prediction using key property features
* phishing_datasets.ows - Detection and classification of phishing URLs and web content
* sentiment_analysis_jul_7.ows - Sentiment analysis pipeline applied to text data
* spamtext.ows - SMS/text spam detection using natural language processing
* telecom_customer_churn.ows - Customer churn prediction for a telecom provider
* youtube_phishing_datasets.ows - Phishing detection analysis applied to YouTube-related datasets

## 📖 About This Repository

This repository serves as a portfolio of my work and a practical reference for Orange's data analysis and visualization capabilities. All projects were built using Orange's visual programming environment, exploring real-world datasets across fintech, cybersecurity, and NLP domains.

## 🛠️ Requirements

Orange Data Mining (version 3.x or later)
Relevant Orange add-ons: Text Mining, Educational

## 🚀 Getting Started

Install Orange from the official website
Clone or download this repository
Open any .ows file directly in Orange
